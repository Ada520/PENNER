EgoSet Data (WSDM 2016)

Original Paper: EgoSet: Exploiting Word Ego-networks and User-generated Ontology for Multifaceted Set Expansion

Authors: Xin Rong, Zhe (Shirley) Chen, Qiaozhu Mei, Eytan Adar

Wikipedia Ontology Lists

  • fully-parsed-wikilists-unigram.txt.zip (2.9 MB) - Wikipedia ontology lists (unigrams)

    Each line represents an ontology list and is comprised of two fields. The first field is the identifier of the list, containing information about how the list is derived. The second field is a JSON array, containing the items of the list. Only the top 30,000 frequent unigrams are included.

  • fully-parsed-wikilists-ngram.txt.zip (6.3 MB) - Wikipedia ontology lists (ngrams)

    The format is the same as above. Only the top 50,000 frequent n-grams (n up to 5) are included. An unfiltered (much larger) list collection is available upon request.

Ground Truth Data

  • gt-classes.txt (80 KB) - Minimum Required Semantic Classes (MRSC)

    There are 89 classes. Each line represents a class. The fields are tabular separated. The first field is the class identifier. The second field is the class name. The remaining fields are the elements in the class.

  • gt-queries.txt (5 KB) - multifaceted queries

    There are 150 queries. Each line represents a query. The fields are tabular separated. The first field is the query (with space-separated terms). The remaining fields are the MSRC identifiers for the query.

Note: to use the ground truth data for evaluation, an alias map or human annotation is required to determine whether a term returned by the system matches any term in one of the MSRCs in the ground truth.
